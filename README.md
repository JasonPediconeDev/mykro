# Mykro

**Micro-donate a percentage of your online gambling winnings to charity.**

Mykro is a platform that lets users automatically donate a small percentage of their gambling winnings to the causes they care about. It works two ways:

1. **Direct Platform Integration** — For gambling platforms that integrate the Mykro SDK, donations happen in real-time on every win.
2. **Bank Auto-Detection via Plaid** — For any platform, users connect their bank account and Mykro automatically identifies gambling deposits and calculates donations.

## For Users (B2C)

- Connect your gambling platforms or bank account
- Choose your charities and set your donation percentage
- Track your impact with a real-time dashboard
- Get tax receipts at year-end

## For Platforms (B2B)

- **React SDK**: `npm install @mykro/react` — Drop-in widget and provider components
- **Node SDK**: `npm install @mykro/node` — Server-side integration
- **REST API**: Full-featured API for custom integrations
- **Webhooks**: Real-time event notifications

## Tech Stack

- React + Vite
- Tailwind CSS
- React Router
- Recharts (dashboard analytics)
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```
